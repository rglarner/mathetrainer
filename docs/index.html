Mathetrainer

Viel Spass mit dem Mathetrainer! Version 1.0.2 vom 22.10.2025
Das Tool ist völlig kostenlos und es werden keinerlei Aktivitäten aufgezeichnet oder ausgewertet.

Als App speichern – kurze Anleitung

  1. Android (Chrome): Seite öffnen → Menü (drei Punkte oben rechts) → "Zum Startbildschirm" / "Add to Home screen".
  2. iPhone / iPad (Safari): Seite öffnen → Teilen‑Icon → "Zum Home‑Bildschirm" auswählen.
  3. Desktop (Chrome / Edge): Seite öffnen → Adressleiste: Install‑Symbol (⤓) oder Menü → "Installieren".

Die App startet danach wie eine normale App und kann auch offline arbeiten (für bereits geladene Seiten). Bei Updates einfach die Seite neu laden. Keine Anmeldung nötig, keine Daten werden übertragen.
